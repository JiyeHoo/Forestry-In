# 描述
Android端设计，林业信息录入系统。
实现登录、注册、主页、侧滑栏、地图、信息录入界面的设计。

# 需要修改的地方
地图使用了高德LBS，需要修改成自己的Key，不修改就是我的Key，可能会无法使用。
